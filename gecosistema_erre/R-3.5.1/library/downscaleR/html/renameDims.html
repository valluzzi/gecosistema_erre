renameDims {downscaleR}R Documentation

Set the 'dimensions' attribute

Description

Sets the 'dimensions' attribute of model out Data objects after downscaling

Usage

renameDims(obs, multi.member)

Arguments

obs

A observations object

multi.member

Logical indicating if simulation data is a multimember

Value

A character vector indicating the dimensions of the output object
