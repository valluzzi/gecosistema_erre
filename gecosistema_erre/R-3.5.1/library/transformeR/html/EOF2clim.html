EOF2clim {transformeR}R Documentation

PCA object EOFs to climatology grid

Description

Converts a prinComp output to a climatology of EOFs. Internal of plotEOF.

Usage

EOF2clim(prinCompObj, ind.var, member, n.eofs)

Arguments

prinCompObj

PCA object

ind.var

index position of the target variable

member

index position of the member whose EOFs are to be displayed

n.eofs

Number of EOFs to be displayed (from 1 to n.eofs)

Value

A climatological grid. Note that EOFs are treated as members
