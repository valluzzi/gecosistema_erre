expandIrregularCoords {transformeR}R Documentation

Expand irregular grid coords

Description

Expand the coordinates of irregular grids into a 2D matrix of x-y coordinates

Usage

expandIrregularCoords(irregularXYcoords)

Arguments

irregularXYcoords

And irregular xyCoords element of an irregular grid

Value

a 2D matrix of x-y coordinates
