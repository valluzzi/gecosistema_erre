reorderStation {transformeR}R Documentation

Reorder loc dimension in irregular grids

Description

Retrieves a grid that is with the 'loc' dimension reordered. Multimember multigrids are supported. Subroutine of subsetGrid.

Usage

reorderStation(grid, axis = c("x", "y"))

Arguments

grid

Input irregular grid to be subset (possibly a multimember/multigrid).

lon

Logical to order data according to longitudes.

lat

Logical to order data according to latitudes.

Value

An irregular grid (or multigrid).
